# Key System Project

## Backend Setup
1. `cd backend`
2. `npm install`
3. Copy `.env.example` to `.env` and set your database URL + password
4. `npm start`

## Frontend Setup
1. `cd frontend`
2. `npm install`
3. `npm start`

## Features
- Add, revoke, and list keys from dashboard API
- Test page validates keys (one-time use)
- Secure password-protected dashboard
